# Freedman's Bank: Forensic Data Analysis (1872-1874)
### Обзор проекта
Анаализ исторических данных **Freedman's Saving Bank** накануне финансовой финансовой паники 1873 года (первый экономический кризис в истории США).
**Задача** Проверить гипотезу о "набеге вкладчиков" и выявить реальные причины банкротства и сегментировать клиентскую базу.
### Технический стек
* **Python:** Pandas, Numpy, Matplotlib
* **Concepts:** Data Cleaning, ETL, Time Series Analysis, Cohort Analysis, RFM Segmentation

### Главные инсайты
1.  **Миф о панике** Анализ `Running Total` (накопительного итога) показал, что **массового оттока вкладов не было**. График баланса рос вплоть до закрытия банка. Вкладчики сохраняли доверие, проблема была на стороне управления активами.
2.  **Аномалии и Должники:** Выявлены клиенты с критическим овердрафтом (например, Alfred McCarock, долг ~$1200), чьи операции носили подозрительный характер перед крахом.
3.  **Демография:** Самый высокий средний чек обнаружен у категории "Organizations/Societies" (Церкви и общины), несмотря на их малочисленность по сравнению с частными лицами.
4.  **Retention:** Гистограмма LTV показала, что банк имел ядро лояльных клиентов ("аксакалов"), которые оставались с банком более 5 лет (Isaiah Nichols: 2376 дней).
